# 3DForge

# Description: 3DForge is an open-source 3D animation editor that runs entirely in the browser. Create scenes, animate rigs, control cinematic cameras, and export animations without installing heavy software.

## Features

* Create and edit 3D objects
* Modern and responsive interface
* Fast local rendering
* Cross-platform support
* Extensible architecture
* Open-source development

## Installation

```bash
git clone https://github.com/yourusername/3DForge.git
cd 3DForge
npm install
npm run dev
```

## Building

```bash
npm run build
```
